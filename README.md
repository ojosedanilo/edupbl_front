# EduPBL

## 🖥️ Tecnologias do Front-End

- **Node.js** -> runtime JS
- **pnpm** -> gerenciador de pacotes
- **Vite** -> bundler/dev server
- **React + TypeScript** -> UI
- **SWC** -> compilador
- **TanStack Query** -> estado servidor
- **React Router** -> rotas
- **TailwindCSS** -> estilos

## 🧩 Camadas

|           Coluna 1           |
| :--------------------------: |
|    React + TS + Tailwind    |
|              ↓              |
| TanStack Query (HTTP State) |
|              ↓              |
| FastAPI (Rules / Validation) |
|              ↓              |
|   SQLAlchemy (Persistence)   |
|              ↓              |
|          PostgreSQL          |
